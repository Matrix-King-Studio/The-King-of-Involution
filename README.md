**退出机制**：如果超过3次未参加分享会（❌），则视为退出。如果超过3个周期都未分享，则也视为退出。当然，如果后续不想继续参加，直接退群就可以啦，不强求~

**要求**：

1. 提前一周准备好PPT，可以发给刘兆峰看一下，然后针对性修改；
2. 每个人至少向分享人提一个问题，如果不问题，分享人可以反问一个问题；
3. 会上没有解答出来的问题，会后学习并整理，发到群里。
